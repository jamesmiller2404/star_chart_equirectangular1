## d3-celestial Milky Way outlines

`d3-celestial-mw.json` is vendored from d3-celestial:

- Source: https://github.com/ofrohn/d3-celestial/blob/master/data/mw.json
- Upstream format docs: https://github.com/ofrohn/d3-celestial/blob/master/data/readme.md
- License: BSD-3-Clause; see `LICENSE-d3-celestial`

The file is GeoJSON with five `MultiPolygon` features (`ol1` through `ol5`) that represent Milky Way outline brightness steps.
